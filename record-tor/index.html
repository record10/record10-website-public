Installation

macOS
  1. 1 Download RecordTor_x64.dmg from the latest release.
  2. 2 Open the .dmg file and drag RecordTor into your Applications folder.
  3. 3 The app is unsigned. macOS will block it on first launch. Run this command in Terminal once:
xattr -cr /Applications/RecordTor.app
Windows
  1. 1 Download RecordTor_x64-setup.exe from the latest release.
  2. 2 Run the installer and follow the on-screen instructions.
  3. 3 The app is unsigned, so Windows SmartScreen may show a warning. Click "More info" and then "Run anyway".
Linux
  1. 1 Download record-tor_amd64.AppImage or record-tor_amd64.deb.
  2. 2 For AppImage, make it executable and run:
chmod +x record-tor_amd64.AppImage && ./record-tor_amd64.AppImage

Or install via .deb:

sudo dpkg -i record-tor_amd64.deb
Android
  1. 1 Download RecordTor.apk from the latest release.
  2. 2 Open the APK file on your device.
  3. 3 If prompted, enable "Install from unknown sources" in your device settings.
